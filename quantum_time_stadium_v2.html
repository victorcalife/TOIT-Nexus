TQL: Linguagem de Consulta em Português

❌ SQL Tradicional

SELECT SUM(sales.value) as total,
       AVG(sales.value) as average,
       COUNT(*) as quantity
FROM sales
WHERE date >= '2025-01-01'
  AND date <= '2025-01-31'
GROUP BY product_id
ORDER BY total DESC
LIMIT 10;
-- Complexo e em inglês

✅ TQL em Português

vendas_mes = SOMAR valor DE vendas 
    ONDE data = MES(0);

ticket_medio = MEDIA valor DE vendas 
    ONDE data = MES(0);

MAX(10) produtos POR vendas;

TQL na Prática: Casos Reais

💼 Dashboard de RH em 5 linhas:

funcionarios = CONTAR funcionarios ONDE status = "ativo";
salario_medio = MEDIA salario DE funcionarios;
turnover = demissoes / funcionarios * 100;

DASHBOARD "RH":
    KPI funcionarios TITULO "Total Funcionários";
    KPI salario_medio TITULO "Salário Médio", MOEDA R$;
    KPI turnover TITULO "Turnover", FORMATO %;

📈 Análise de Vendas Temporal:

# Comparativo mensal automático
vendas_atual = SOMAR valor DE vendas ONDE data = MES(0);
vendas_anterior = SOMAR valor DE vendas ONDE data = MES(-1);
crescimento = (vendas_atual - vendas_anterior) / vendas_anterior * 100;

# Top produtos do mês
MAX(5) produtos POR vendas ONDE data = MES(0);
